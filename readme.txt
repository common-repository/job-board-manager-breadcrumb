### Job Board Manager - Breadcrumb by http://pickplugins.com

* [Live Demo!&raquo;](http://www.pickplugins.com/demo/job-board-manager/)
* [Get Job Board Manager !&raquo;](https://wordpress.org/plugins/job-board-manager/)

Automatically display breadcrumb on single job page. 

There is no settigns for this addon.



== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>Job Board Manager - Breadcrumb</strong>" activate it.<br />
